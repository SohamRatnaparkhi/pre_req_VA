PRE-REQUISTES to use the Voice Assistant!

Following are the pre-requisites to be installed in the system to use Voice Assistant through Terminal Window

  1. Python3 should be installed in the system. Click to download

  2. Following libraries and packages are to be installed. The syntax for library installation of a library and a package is: - installation of a library and a package
    1. pip install (library/package_name) or
    2. pip3 install (library/package_name)
    3. Enter the following commands
    1. pip install colorama
    2. pip install rich
    3. pip install pyttsx3
    4. pip install Date Time
    5. pip install SpeechRecognition
    6. pip install docx
    7. pip install firz
    8. pip install gTTS
    9. pip install playsound
    10. pip install pywin32
    11. pip install wikipedia
    12. pip install web browser
